# Unity Dialogue System - Iphigames

## About

This is a simple Unity dialogue system prototype created as a technical assessment for the IPHIGAMES interview process.

## Unity Version

This project was created with Unity version `6000.3.6f1`.

## How To Test

To test the project:

1. Open the project in Unity.
2. Load the main scene at `Assets/Scenes/Main.unity`.
3. Press Play in the Unity Editor.

## Current Limitation (Simplicity Tradeoff)

Because this project is intentionally simple, creating multiple dialogue entries for the same speaker is not very convenient right now.
You must manually set the speaker image and speaker name for each entry.

In a real production game, I would create `ScriptableObject` assets for each possible speaker and reuse them across dialogue entries.
